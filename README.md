# Autonomous-Car

Self-driving RC-scale car build. Personal engineering project — Summer 2026.

## Project Overview

Design, fabricate, and program a small-scale autonomous vehicle from scratch. The car will use onboard sensors and a microcontroller to navigate without human input. Combines mechanical design, electronics, and software into one integrated build.

## Planned Tech Stack

- **CAD / Mechanical Design:** Fusion 360
- **Microcontroller:** Arduino (with possible Raspberry Pi for higher-level control)
- **Programming Languages:** C/C++ (Arduino), Python (control logic)
- **Sensors:** Ultrasonic, IMU, possibly camera module
- **Version Control:** Git / GitHub

## Project Phases

- [ ] Phase 1: Chassis design in Fusion 360
- [ ] Phase 2: Component selection and parts ordering
- [ ] Phase 3: Electronics breadboard prototype
- [ ] Phase 4: Motor control and basic movement
- [ ] Phase 5: Sensor integration
- [ ] Phase 6: Autonomous navigation logic
- [ ] Phase 7: Physical assembly and integration testing
